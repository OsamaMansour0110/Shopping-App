**Language:** Java
**Description:**
A comprehensive e-commerce Android application with features for both users and administrators.
**Key Features:**
- **User Features:**
    - User Authentication (Login, Sign-up, Password Recovery).
    - Product browsing by categories.
    - Shopping Cart functionality.
    - Location services (Google Maps integration).
- **Admin Features:**
    - Dashboard for managing items and categories.
    - Analytics visualization (Bar charts, Pie charts, Sales graphs).
    - Feedback and Report management.
- **Database:** Utilizes SQLite for local data management.
